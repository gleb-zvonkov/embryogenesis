# Embryogenesis 

An interactive web app that visualizes embryonic development from fertilization through week 8.

## Features

- Scroll-based timeline (Day 0 → Week 8)
- Custom navigation indicator
- Hoverable terms with definitions
- Clickable terms that open related Google searches
- Smooth animations and UI interactions

## Tech Stack

- React + TypeScript
- Vite
- CSS

## Notes
Most images used in this project are fully or partially generated using the DALL·E model.

## Deployed on Github Pages
